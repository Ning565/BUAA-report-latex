# BUAA-report-latex
这里是北航（BUAA）课程报告`Latex模版`（改编自XDUreport:西安电子科技大学实验报告 LaTeX 模板），在课程没有明确报告模版要求时可使用本模版，无痛编译，一键使用
1. 请直接git/download，文件结构为
     - BUAA_Report
         - 【请先安装字体，否则报错】Fonts
         - cover （存放北航logo和name）
         - img（存在文档图片）
         - BUAAreport.sty（latex配置文件，包括页眉页脚、公式图表、代码目录和封面等的配置）
         - BUAAreport-Demo.tex 编写tex文件，生成报告
         - BUAAreport-Demo.pdf pdf模版演示
2. 打开tex文件后，在开头按照注释修改为自己的信息，编译运行即可，在Windows 环境本地完美运行，使用引擎 `TeXstudio 4.6.3`  ，编译方式为 `xelatex`，默认字体方式为 `UTF-8`，暂未有Overleaf版本
3. 本模板谨按较为私人化，以自用为主，**如果老师有明确要求，请勿使用此模板！！！**
4. 若对样式有额外需求或不满之处，请自行修改 BUAAreport.sty 配置类文件。
